# Aniview Documentation Website

This directory contains the Docusaurus documentation site for Aniview.

## Installation

```bash
npm install
```

## Local Development

```bash
npm run start
```

This starts a local development server. Most documentation changes are reflected without restarting the server.

## Build

```bash
npm run build
```

This generates static content into `build/`.

## Deployment

The site is configured for GitHub Pages at `/aniview/`. Maintainers can deploy with:

```bash
GIT_USER=<github-username> npm run deploy
```
